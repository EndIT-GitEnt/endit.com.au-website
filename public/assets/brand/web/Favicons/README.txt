End iT favicon pack

Included variants:
- transparent: orange symbol-only favicon assets on transparent background
- dark: orange symbol-only favicon assets on dark square background

Typical files included for each variant:
- favicon.ico (16, 32, 48, 64)
- favicon-16x16.png
- favicon-32x32.png
- favicon-48x48.png
- favicon-64x64.png
- favicon-96x96.png
- favicon-128x128.png
- favicon-256x256.png
- apple-touch-icon.png plus older Apple touch sizes
- android-chrome-192x192.png
- android-chrome-512x512.png
- mstile-150x150.png
- site.webmanifest
- browserconfig.xml

Suggested default for a website:
- Use the transparent variant if you want the symbol alone without a square background.
- Use the dark variant if you want a stronger boxed favicon look.
